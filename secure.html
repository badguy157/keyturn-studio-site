How it works Back to onboarding

Option A — 1Password Send (recommended)

  1. Open 1Password Send.
  2. Choose Text. Paste credentials (e.g., “Registrar: Namecheap — user: … / pass: …”).
  3. Click Link options → set Expiry (e.g., 7 days) and Max views (1).
  4. Click Create link, then copy the link.
  5. Send the link to us at hello@keyturn.studio.

Notes: End-to-end encrypted. No account needed for you. We’ll open once, store in our vault, and confirm receipt.

Option B — Bitwarden Send

  1. Open Bitwarden Send.
  2. Create a Text Send, set Expiration and Max access count (1), optionally add a Password.
  3. Copy the link and email it to hello@keyturn.studio.

Also end-to-end encrypted. Password option available.
